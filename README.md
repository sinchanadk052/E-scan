# E-scan
🔍 Turn images into text instantly — An OCR-powered document scanner built with EasyOCR 
🚀 Digitize, manage, and secure your documents with ease — An AI-powered EDMS featuring OCR scanning, role-based access, and seamless document lifecycle management.
⚡ Web-based EDMS | Python + Flask + MongoDB | LDAP login, role-based access, document scanning (OCR), and metadata-driven storage for enterprise-grade document control.

## ✨ Features
🔐 **Secure Login** – Supports both **LDAP** and application-level authentication.  
👥 **Role-Based Access** – Define user roles (Admin, Manager, General User) with tailored permissions.  
📄 **Document Lifecycle Management** – Create, edit, enable/disable, and archive documents.  
🏷 **Metadata Tagging** – Store and retrieve documents using rich metadata fields.  
📷 **Document Scanning (OCR)** – Scan and digitize physical documents with automatic tagging.  
⚡ **Centralized Repository** – Fast and reliable storage, retrieval, and search of documents.  
📊 **Audit & Security** – Encrypted storage, activity tracking, and role-based data protection.  

## 🛠️ Tech Stack
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MongoDB  
- **Authentication:** LDAP + Application-based login  
- **OCR:** EasyOCR (for document scanning)
